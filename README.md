(Optimal, in https://github.com/hqqaazz/orderedEnchanting-Demo/wiki/Optimal-enchantment-order)

2. Looting III+Sharpness V/Smite V/Bane of Arthropods V
3. (Fire Aspect II+Unbreaking III) + (Knockback II+Mending)
